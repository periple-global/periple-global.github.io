# 📦 제품 이미지 업로드 가이드

## 폴더 구조 — 카테고리별로 사진을 넣어주세요

```
products-new/
├── 01_skincare/
│   ├── bottle/         ← 토너·로션·세럼 보틀, 글래스/PET 보틀
│   ├── jar/            ← 크림 자, 마스크팩 자, 코튼 패드 자
│   ├── airless/        ← 진공 에어리스 보틀·자
│   ├── tube/           ← 페이셜·핸드·바디 튜브, 친환경 PCR/뱀부
│   ├── dropper/        ← 스포이드 보틀
│   └── pump/           ← 대용량 펌프 보틀
├── 02_makeup/
│   ├── pen-stick/      ← 립스틱, 트위스트/클릭 펜, 컨실러 펜
│   ├── tube/           ← BB·파운데이션·아이크림 튜브
│   └── compact/        ← 쿠션, 컴팩트, 파우더
├── 03_special/
│   ├── massage-tube/   ← 마사지 볼·5비드 튜브
│   ├── roller-applicator/ ← 롤러 보틀, 두피·아이 어플리케이터
│   └── syringe-pen/    ← 시린지 앰플, 멀티노즐
└── _placeholders/      ← 현재 사이트가 임시로 쓰는 PDF 추출 이미지 (건드릴 필요 X)
```

각 폴더 안에 **`_README.txt`** 가 있어요. 어떤 사진을 넣으면 좋은지 한 줄 안내가 적혀있습니다.

## 📝 파일명 규칙

**자유롭게 지으셔도 됩니다.** 다만 한 카테고리 안에서 헷갈리지 않게 의미 있는 이름이면 좋아요.

좋은 예:
- `glass-set-30-100ml.jpg`
- `eco-tube-bamboo-50ml.jpg`
- `airless-roller-eye-cream.jpg`
- `twist-pen-lip-3ml.jpg`

피해야 할 이름:
- `IMG_2541.jpg` (의미 없음)
- `KakaoTalk_2025-01-02_xxxxx.jpg` (자동 이름)

## 🚀 작업 흐름

### 1단계 (지금) — **카테고리 폴더에 사진 넣기**

각 카테고리 폴더에 가지고 계신 사진들을 의미 있는 이름으로 저장.

> **모든 SKU를 한 번에 다 넣지 않아도 OK** — 시작은 카테고리당 5~10장 정도, 나중에 더 채워가도 됩니다.

### 2단계 — **메타정보 (용량 / 소재 / 특징) 입력**

사진 파일명에 일부 정보를 담거나(예: `glass-30ml-pump-toner.jpg`), 또는 별도로 알려주시면 페리플 측에서 입력합니다.

가장 편한 방법: 각 사진에 대해 노션/엑셀 표에:

| 파일명 | 용량(ml) | 소재 | Usage | 특징 |
|---|---|---|---|---|
| glass-set-30-100ml.jpg | 30/50/100 | Glass | Toner, Serum, Cream | 골드 캡 |

### 3단계 — **자동 카탈로그 갱신**

다 넣으셨으면 페리플 측에 "이미지 업로드 완료" 라고 알려주세요. 아래를 자동 처리합니다:
1. 폴더 스캔해서 모든 사진을 SKU 카드로 등록
2. `products.json` 자동 생성 (현재 42개 → 사용자가 넣은 만큼 자동 확장)
3. 카드 그리드에 즉시 반영

## ❓ "PDF 모든 제품을 사이트에 다 노출하려면?"

현재 카탈로그는 **42개 SKU 큐레이션** 상태입니다. PDF에는 수백 개가 있지만 일부 시리즈로 묶어 대표 카드로 등록한 상황입니다.

**전체 노출 방안:**

### 옵션 A — 사용자가 PDF에서 사진을 따로 추출 (권장)
PDF에서 마음에 드는 모델만 골라 캡처/스크린샷해서 카테고리 폴더에 직접 넣기.
→ **품질·익명성 모두 사용자 통제 가능.**

### 옵션 B — PDF 페이지 자체를 모두 카드로 등록
페리플 측이 405페이지를 모두 카드로 자동 등록. 단 **회사 로고가 페이지 안에 노출**되므로:
- 사용자가 후처리로 로고 부분 마스킹 (Photoshop/온라인 툴)
- 또는 페리플 측이 자동 마스킹 (정확도 ~80%, 일부 수동 보정 필요)

### 옵션 C — PDF에서 회사명·로고 자동 제거 후 카드 등록
페리플 측이 OCR + 이미지 처리로 페이지에서 텍스트·로고를 자동 마스킹한 후 등재.
→ 작업 시간 1~2일 소요.

**어떤 옵션을 원하시는지 알려주시면 진행합니다.**
